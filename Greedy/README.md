
### **그리디(Greedy) 알고리즘이란?**

greedy : 탐욕스러운 -> 탐욕법으로 부르기도한다.

그때그때 좋은것만 고르는 방법

따라서 '**가장**' 이라는 말과 자주 등장하며, '가장' 이라는 말은 정렬 알고리즘과 어울리므로  **정렬과 자주 엮인다.**

문제를 해결할 때 **좋아 보이는것을 택하는 것이 최적의 해를 구할 수 있을때** 그리디로 해결 아이디어를 구현해본다.
